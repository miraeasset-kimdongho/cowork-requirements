# ISSUE-009: 함께 띄운 타인 일정에도 '내 기기 일정에 저장하기' 버튼 노출

> **제품**: Mattermost Calendar 플러그인
> **유형**: `개선`
> **작성자**: 김동호 책임
> **작성일**: 2026-06-18
> **상태**: `Open`
> **관련 PRD**: [PRD-007](../PRD/PRD-007_캘린더-기능-부가-개발.md)

---

## 1. 현상 / 요청 내용

- 본인 일정을 조회하면 토스트 팝업 우측 상단 **`...`** 메뉴에 **"내 기기 일정에 저장하기"** 버튼이 있음.
- 현재 이 버튼은 **본인 일정에만** 노출됨.
- **함께 띄운 다른 사용자 일정에도** 동일하게 노출해달라.
- 의도: 타인 일정도 내 디바이스 캘린더에 저장하고 싶다는 **박홍근 대표님 요청**.

---

## 2. 기대 결과 (To-Be)

- 다른 사용자 일정 조회 시에도 `...` 메뉴에 **"내 기기 일정에 저장하기"** 버튼 노출
- 클릭 시 해당 타인 일정이 사용자 디바이스 캘린더에 저장됨

---

## 3. 재현 경로 (버그일 경우)

- 해당 없음 (기능 개선)

---

## 4. 대상 화면 / 영향 범위

- 대상 화면: 모바일 캘린더 → 함께보기로 띄운 타인 일정 상세 토스트 팝업 `...` 메뉴
- 영향 범위: 모바일 캘린더 사용자 전체 (함께보기 사용자)

---

## 5. Before / After (UI 변경일 경우)

**Before**
> "내 기기 일정에 저장하기" 버튼이 본인 일정에만 노출

**After**
> 함께 띄운 타인 일정에도 동일 버튼 노출, 타인 일정도 내 기기 캘린더에 저장 가능

---

## 6. 구현 메모

- 본인 일정 저장 로직을 타인 일정에도 재사용. 권한/공개범위(sensitivity) 처리 확인 필요.
- 박홍근 대표님 요청 사항.
