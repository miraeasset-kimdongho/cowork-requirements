# ISSUE-012: 함께보기 공유 일정 중복 표시 → 1개로 병합

> **제품**: Mattermost Calendar 플러그인
> **유형**: `개선 요청`
> **작성자**: 권현준 매니저
> **작성일**: 2026-06-26
> **상태**: `Open`
> **관련 PRD**: [PRD-007](../PRD/PRD-007_캘린더-기능-부가-개발.md)
>

---

## 1. 현상 / 요청 내용

- 함께보기로 여러 명을 함께 볼 때, **공유 일정**(한 명이 생성하고 나머지는 참석자로 추가된 동일 일정)이 **참석자 수만큼 중복 표시**됨.
- 예: 10명이 참석하는 회의를 함께보기로 보면 캘린더에 **같은 회의가 10번** 겹쳐 표시.
- 동일한 공유 일정은 **1개의 블록으로만** 표시되게 해달라.

---

## 2. 기대 결과 (To-Be)

- 동일한 공유 일정은 참석자 수와 무관하게 **캘린더에 1개 블록으로만** 표시.
- 일정 색상은 본인 색상으로.
- 병합된 블록에 **참석 인원 표기**(예: `주간회의 · 참석 10`) — 클릭/상세에서 참석자 목록 확인.
- 본인(로그인 사용자) 미 참석 시, 일정 표출 X
- 비공개 마스킹(`isHiddenPrivate`)은 병합 후에도 동일 적용.

---

## 3. 재현 경로

1. 함께보기에 회의 참석자 10명을 추가(한 명이 생성, 나머지 9명은 참석자)
2. 캘린더(주/월 뷰) 조회
3. **(현상)** 동일 회의가 10명 각각의 일정으로 **10개 블록 중복 표시**

---

## 4. Before / After (UI 변경)

**Before**
```
09:00 ┃ 주간회의   (권현준)
09:00 ┃ 주간회의   (김동호)   ← 같은 회의가
09:00 ┃ 주간회의   (김문기)      참석자 수만큼
09:00 ┃ 주간회의   (이동환)      10개 중복 표시
  …   ┃   …  (총 10개)
```

**After**
```
09:00 ┃ 주간회의 · 참석 10      ← 1개 블록으로 병합
        (상세에서 참석자 목록)
```

---

## 5. 체크리스트

- [ ] 동일 공유 일정 병합 표시
- [ ] 일정 색상은 본인 색상으로
- [ ] 참석 인원 수 표기 + 상세에서 참석자 목록
- [ ] 비공개 마스킹 유지(부분 비공개 규칙 포함)
- [ ] 성능(다수 인원·장기간) 확인
- [ ] 데스크탑·(태블릿) 동작 확인
