# ISSUE-008: '함께보기' 토스트 팝업 헤더 문구 변경 및 '등록중인 사용자' 영역 제거

> **제품**: Mattermost Calendar 플러그인
> **유형**: `UI 변경`
> **작성자**: 김동호 책임
> **작성일**: 2026-06-23
> **상태**: `Done`
> **관련 PRD**: [PRD-007](../PRD/PRD-007_캘린더-기능-부가-개발.md)

## 1. 현상 / 요청 내용

- 모바일 캘린더 뷰 우측 상단 **`+ 함께보기`** 버튼을 누르면 뜨는 토스트 팝업에서:
  - 헤더 문구 **"내 캘린더 함께보기"** → **"다른 사용자 일정 같이 띄우기"** 로 변경
  - 하단 **"등록중인 사용자"** 영역을 **아예 제거**
- 6/22 추가, 모바일 내 함께보기 버튼의 팝업에도 문구 변경 적용을 요청합니다 (사용자 경험 일원화를 위해)

---

## 2. 기대 결과 (To-Be)

- 토스트 팝업 헤더가 **"다른 사용자 일정 같이 띄우기"** 로 노출
- 하단 "등록중인 사용자" 영역 미노출
- "등록중인 사용자" 텍스트가 차지하던 **WIDTH 만큼 토스트 팝업 가로 길이 축소**
