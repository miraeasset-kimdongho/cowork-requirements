## 캘린더 플러그인 v1.3.6

### 변경 사항
- ISSUE-001: 일정 장소(location) 필드 추가
  - EWS 캘린더에서 location 파싱 => 상세조회 / 초대 / 리마인더까지 전달
  - 일정 생성/수정 모달에 '장소' 입력 추가
  - 리마인더/초대 본문에 '장소' 추가
- ISSUE-002: 캘린더 헤더 '오늘' 버튼 추가
- ISSUE-004: 일정 알림 DM이 종종 오지 않는 버그 수정 (리마인더 발송 신뢰성)
  - SENT 마킹을 발송 성공 후로 이동 => 노드 크래시/재배포 시 영구 누락 방지
  - 형제 행 봉인 버그 수정 (SENT 행에 막혀 스킵되던 현상)
  - 24시간 초과(stale) 과거 리마인더 자동 정리 => 배포 시 DB 수동 정리 불필요
  - 일시 오류 시 PENDING 유지 후 다음 사이클 재시도
- ISSUE-005: 초대 메시지 'message deleted' 표시 제거
  - 변경/제외/취소/만료 시 삭제 대신 안내 문구로 교체하고 버튼 제거
- 참석자 응답 표시 개선
  - 비주최자가 상세를 열 때 주최자 사본(저장된 calendarID) 기준으로 수락/거절 표시
  - EWS는 참석자 본인 사본에 타 참석자 응답을 Unknown으로만 반환하는 한계 보완
  - 프론트: 일정 참석자 응답 상태(수락/거절/미정/미응답) 뱃지 표시
- 응답 액션 처리 개선
  - 제목 변경(rename) 대응 매칭 (시간+제목 => 시간 => 제목)
  - 일시 오류 시 버튼을 보존하는 재시도 응답
- 종료일 배타경계 수정: CalendarView endTime 항상 +1로 종료일 일정 누락 방지
